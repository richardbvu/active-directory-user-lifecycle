# Active Directory Support Lab

## Overview
This project documents the build, validation, and troubleshooting of a Active Directory lab environment.

The lab includes:
- A Windows Server domain controller
- A domain-joined Windows 11 workstation
- Baseline Group Policy configuration
- User lifecycle and workstation support troubleshooting

This project was documented using an evidence-first workflow, with build evidence, troubleshooting evidence, runbooks, ticket write-ups, PowerShell validation, and resume/interview packaging.

## Skills Demonstrated
- Active Directory user lifecycle management
- Workstation domain join validation
- Group Policy verification
- DNS troubleshooting in a domain environment
- Remote Desktop access troubleshooting
- Domain user authentication troubleshooting

## Project Structure
- **00-Project-Summary** — overview and navigation
- **01-Build-Evidence** — healthy-state build validation
- **02-Troubleshooting-Evidence** — raw notes, screenshots, and command outputs
- **03-Runbook** — troubleshooting procedures
- **04-Ticket-Pack** — polished ticket write-ups
- **05-PowerShell** — command output and validation files
- **06-Resume-and-Interview** — project packaging for job search
